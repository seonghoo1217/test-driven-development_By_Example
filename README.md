# Test-Driven Development : By Example

## 컨벤션

해당 레포지토리에서 요구되는 컨벤션은 다음과 같습니다.

- indent depth 를 최대 2까지만 유지한다.
- 커밋은 기능단위로 커밋을 하며 커밋 표준 컨벤션을 유지한다.
- 함수는 10의 길이를 넘지않으며 한가지 기능만 사용하도록 구현한다.
- 테스트 위주의 기능 개발을 우선시한다.
